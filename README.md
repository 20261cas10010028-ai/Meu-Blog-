# Meu Blog Pessoal

Este projeto é um blog simples desenvolvido para praticar HTML e CSS.

## Tecnologias Utilizadas
- **HTML5**: Estruturação do conteúdo.
- **CSS3**: Estilização e layout.

## Parceria com IA
Este projeto foi desenvolvido em parceria com o **Google Gemini (IA)**. 
A IA auxiliou na criação da estrutura do código, explicações técnicas via comentários e organização do passo a passo.
